# 📊 E-Commerce Sales Dashboard | Microsoft Power BI

## Project Overview

This project was completed during my **DecodeLab Data Analytics Virtual Internship**.

The objective was to design an interactive Power BI dashboard that transforms raw e-commerce sales data into meaningful business insights. The dashboard enables stakeholders to monitor key performance indicators (KPIs), evaluate sales performance, and make informed business decisions through interactive visualizations.

---

# Business Problem

Businesses generate large amounts of transactional data every day, but without proper visualization and analysis, it is difficult to identify trends, monitor performance, and make strategic decisions.

This project addresses that challenge by developing an interactive dashboard that highlights revenue, sales trends, product performance, customer order patterns, and marketing channel effectiveness.

---

# Tools Used

* Microsoft Power BI
* Power Query
* DAX
* Microsoft Excel (Dataset Preparation)

---

# Dataset

The dataset contains **1,200 e-commerce transactions** spanning **2023–2025** and includes:

* Order ID
* Date
* Product
* Quantity
* Unit Price
* Total Price
* Order Status
* Payment Method
* Referral Source
* Items in Cart

---

# Key Insights

The dashboard revealed that:

* Total revenue exceeded **$1.26 million** from **1,200 orders**.
* Chairs, Printers, and Laptops generated the highest revenue.
* Instagram was the best-performing referral source in both revenue and order volume.
* Cancelled and Returned orders accounted for over **41%** of total orders, indicating opportunities to improve order fulfillment.
* Revenue was highest in **2023**, with **2024** showing a decline. Data for **2025** represents only part of the year.
* Sales peaked during **May** and **June**, suggesting seasonal purchasing patterns.
